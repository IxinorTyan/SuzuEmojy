# SuzuEmojy 使用说明书

感谢使用 SuzuEmojy！

## 1. 安装与启动(便携版可以无视)
- **安装依赖**：首次使用请双击 `SuzuEmojy` 开始安装依赖。如果安装失败，可以尝试运行旁边的 `依赖装不上,没招了你就试试点这个吧,记得附上报错日志.bat`。
- **启动方式**：启动后，软件会静默运行在后台，**菜单被置于系统托盘图标内**（即屏幕右下角，QQ/微信图标所在的位置）。

## 2. 核心功能：快捷唤出
- **快捷键**：默认使用 `Ctrl + Shift + E` 唤出主界面。这是软件的核心功能，让你在需要时能瞬间唤起！
- **托盘操作**：左键单击托盘图标可唤出主界面，右键单击可展开菜单进入**设置页面**。

## 3. 界面布局与操作

### 3.1 侧边栏（分类管理）
左侧是你的表情分类夹，默认包含“全部表情”等。
- **拖动排序**：支持直接拖动分类夹进行排序。
- **右键菜单**：右键分类夹可弹出子菜单，进行重命名、删除等操作。
- **自定义图标**：你可以将某个表情设置为分类夹的专属标签图片。
- **缩略图模式**：双击“全部分类”可进入标签缩略图模式，此模式下可使用 `Ctrl + 鼠标滚轮` 调节缩略图大小。

### 3.2 表情仓库（主展示区）
右侧是表情展示区，支持以下便捷操作：
- **快速导入**：支持直接**拖入图片**或**复制粘贴**导入表情。支持直接拖入文件夹进行快速导入.
- **特殊导入**:   把图片另存为到目录下的(\bin)\data\inbox就能自动导入,此方法适用于webm(通常不使用这个导入方法,可以说几乎是针对telegram动图设计的方法...)
- **拖动整理**：
  - 在仓库内拖动图片可自定义排序。
  - 将图片直接拖动到左侧的分类夹上，即可快速完成分类（推荐从“未分类”开始整理）。
- **动图预览**：将光标悬停在表情缩略图上超过 500ms（可在设置中调整），即可预览动图。
- **缩放视图**：使用 `Ctrl + 鼠标滚轮` 可对缩略图进行全局缩放。
- **右键菜单**：右键表情可进行细致管理，包括添加 Tag（标签）、设为分类图标等。

### 3.3 搜索,筛选与批量操作
- **搜索功能**：右上角的搜索框可以快速查找你添加过 Tag 的表情。
- **筛选功能**:   搜索旁边有筛选选项,可以进行一定的筛选辅助更好的管理
- **批量操作**：点击搜索框左侧的图标进入批量操作模式。支持：
  - 批量删除
  - 批量导出
  - 批量添加到子分类夹
  - 批量移出子分类夹（仅在子分类夹内可用）

## 4. 个性化设置
在系统托盘右键菜单中进入“设置”，你可以：
- 决定是否保持主界面始终置顶。
- 调整侧边栏图标大小、悬停预览延迟等参数。
- 修改全局唤出快捷键。
